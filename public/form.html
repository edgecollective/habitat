Bayou-CO2: Cooperative CO2 Sensor Data

Bayou-CO2 is designed to be a simple way of capturing, storing, and retrieving CO2 sensor data. Its creation was motivated by recent work (e.g. by Jimenez et al.) indicating that monitoring indoor CO2 levels can (in some contexts) help to assess the risk of airborne SARS-CoV-2 transmission.

Open Code / Open Data. All the code for Bayou-CO2 is free and open source. You can download all historical data as CSV or JSON. More information on how to send and receive data to/from Bayou is provided when you create a new feed, below. Source code and additional background can be found here.

Open Hardware. Bayou can receive data from any device capable of making HTTP POST requests. Many inexpensive microcontrollers can be programmed to do this (when connected to suitable CO2 sensors). You can find some guidelines on doing this here.
