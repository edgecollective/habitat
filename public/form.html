Bayou: Cooperative Sensor Data

Bayou is intended as a simple way of storing and accessing sensor data (e.g. for use in farming, air quality, flood detection, etc.).

All the code for Bayou is free and open source. (More about Bayou here.)

You can create a new data feed, which will then display the necessary credentials for viewing and adding to your new data feed:
